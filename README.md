# ElifTech Food Delivery App

## [Live Demo](https://www.notion.so/)

## Local Setup

- Clone this repo;
- In repo directory run `npm i`;
- In repo directory run `npm run dev`;

## Env variables

```
VITE_SITY_KEY='your_google_recaptcha_site_key'
VITE_API_URL='your_api_url'
```
